Steering Committee

The Steering Committee of the Reversible Computation conference is responsible for the following tasks:

  1. Select the Program Chairs and the Organizers for the next edition of Reversible Computation conference
  2. Advise the Program Chairs and the Organizers on matters related to the conference
  3. Review regularly the format of Reversible Computation conference and the type of publication venue
  4. Manage and maintain Reversible Computation web pages
  5. Answer enquiries regarding Reversible Computation conference and its web pages
